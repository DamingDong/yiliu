## 用户故事

### 故事 1: 快速记录想法

**角色**: 职场人士
**场景**: 开会时想到一个重要点子
**愿望**: 快速记录下来，事后可以查看
**价值**: 不遗漏任何重要想法

```
场景描述:
小明在开会时突然想到一个产品创意，想要记下来。
但是打开笔记软件 -> 新建 -> 输入 -> 保存 太慢了，
担心错过发言内容。

使用忆流:
1. 打开应用（快捷键或托盘）
2. 直接输入想法
3. 回车保存
4. 继续开会

结果:
- 3 秒内完成记录
- 不会错过会议
- 想法被完整保存
```

**验收标准**:

- [ ] 输入框始终可见
- [ ] 回车键可提交
- [ ] 保存后有明确反馈
- [ ] 记录时间 < 3 秒

---

### 故事 2: 查找之前的笔记

**角色**: 职场人士
**场景**: 想不起来之前记录的一个想法
**愿望**: 能快速找到之前的笔记
**价值**: 知识可复用

```
场景描述:
小明一周前记录了一个关于产品优化的想法，
但是记不清具体内容了，只记得关键词。

使用忆流:
1. 点击搜索框
2. 输入关键词"优化"
3. 看到相关笔记列表
4. 找到目标笔记

结果:
- 快速找到相关笔记
- 语义搜索理解意图
- 关键词匹配作为补充
```

**验收标准**:

- [ ] 搜索框在首页可见
- [ ] 语义搜索返回相关结果
- [ ] 显示匹配度百分比
- [ ] 无结果时有关键词回退

---

### 故事 3: 查看笔记列表

**角色**: 个人用户
**场景**: 想回顾最近的笔记
**愿望**: 看到所有笔记的概览
**价值**: 了解知识沉淀情况

```
场景描述:
周末想要回顾一周记录的笔记，看看都记录了什么。

使用忆流:
1. 打开应用
2. 默认显示笔记列表
3. 上下滑动浏览
4. 点击查看详情

结果:
- 笔记按时间倒序排列
- 显示预览和标签
- 了解知识沉淀情况
```

**验收标准**:

- [ ] 打开应用默认显示笔记列表
- [ ] 显示笔记预览（50 字内）
- [ ] 显示时间、标签
- [ ] 支持下拉刷新

### 故事 6: 导出备份

**愿望**: 导出为通用格式
**价值**: 数据可迁移、可备份

```
场景描述:
小明想要把笔记导出备份，以防数据丢失。

使用忆流:
1. 打开设置/导出
2. 选择导出格式
3. 选择保存位置
4. 导出完成

结果:
- 导出为 Markdown 格式
- 包含所有笔记内容
- 可用其他软件打开
```

**验收标准**:

- [ ] 导出入口在设置中
- [ ] 支持 Markdown 格式
- [ ] 导出包含所有笔记
- [ ] 可选择保存位置

### 故事 8: 系统托盘快速记录

**角色**: 职场人士
**场景**: 在其他应用中想要快速记录
**愿望**: 通过托盘图标快速打开
**价值**: 无需切换应用

```
场景描述:
小明正在写文档，想要临时记录一个想法。

使用忆流:
1. 点击托盘图标
2. 应用窗口弹出
3. 输入想法
4. 最小化/关闭

结果:
- 托盘图标始终可见
- 点击快速打开
- 最小化到托盘
```

**验收标准**:

- [ ] 系统托盘有图标
- [ ] 点击托盘打开窗口
- [ ] 支持最小化到托盘
- [ ] 支持全局快捷键
