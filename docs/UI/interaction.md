## 二、各页面交互

### 2.1 即时灵感

**入口**: 左侧导航「即时灵感」

**功能**:
- 对话式输入框
- 发送消息（Enter 键或点击发送按钮）
- AI 自动回复并添加标签
- 消息历史展示
- 快捷键 ⌘K 快速聚焦

**交互流程**:
```
用户输入 → 发送 → 显示用户消息 → AI 处理 → 显示回复 → 保存到笔记列表
```

**交互元素**:
| 元素 | 行为 |
|------|------|
| 输入框 | 支持多行，Enter 发送，Shift+Enter 换行 |
| 发送按钮 | 点击发送 |
| 消息气泡 | 展示历史记录 |
| AI 回复 | 自动生成标签 |

---

### 2.2 写笔记

**入口**: 左侧导航「写笔记」

**功能**:
- 富文本编辑器
- 工具栏（加粗、斜体、标题、列表、链接）
- 字数统计
- 保存笔记

**交互流程**:
```
打开编辑器 → 输入内容 → 格式化（可选）→ 保存 → 添加到笔记列表
```

**交互元素**:
| 元素 | 行为 |
|------|------|
| 工具栏按钮 | 点击应用格式 |
| 编辑区域 | contenteditable，支持富文本 |
| 字数统计 | 实时更新 |
| 保存按钮 | 点击保存到笔记 |

---

### 2.3 知识库

**入口**: 左侧导航「知识库」

**功能**:
- 分类卡片展示
- 点击分类过滤笔记
- 跳转到搜索页面

**交互流程**:
```
查看分类 → 点击分类 → 跳转到语义搜索并自动填入关键词
```

---

### 2.4 语义搜索

**入口**: 左侧导航「语义搜索」

**功能**:
- 搜索输入框
- 实时搜索
- 搜索结果列表
- 点击结果查看详情

**交互流程**:
```
输入关键词 → 实时搜索 → 显示结果列表 → 点击查看详情
```

---

### 2.5 导出备份

**入口**: 左侧导航「导出备份」

**功能**:
- Markdown 导出
- JSON 导出
- WebDAV 同步配置

**交互流程**:
```
选择导出格式 → 点击导出 → 下载文件
```

---

### 2.6 笔记列表/详情（右侧面板）

**入口**: 右侧固定面板

**功能**:
- 列表视图：展示所有笔记
- 详情视图：展示单条笔记内容
- 切换视图
- 编辑/删除笔记

**交互元素**:
| 元素 | 行为 |
|------|------|
| 笔记卡片 | 点击查看详情 |
| 列表/详情切换 | 点击切换视图 |
| 编辑按钮 | 跳转到写笔记页面，加载内容 |
| 删除按钮 | 确认后删除笔记 |

## 三、快捷键

| 快捷键 | 功能 |
|--------|------|
| ⌘K / Ctrl+K | 聚焦即时灵感输入框 |
| Enter | 发送消息 |
| Shift+Enter | 换行 |
| Escape | 取消输入 |

---

## 四、数据流

```
用户输入
    ↓
前端处理 (输入验证)
    ↓
业务逻辑 (创建/搜索/删除)
    ↓
数据存储 (LibSQL)
    ↓
向量处理 (@xenova/transformers)
    ↓
UI 更新
```

---

## 五、状态管理

| 状态 | 说明 |
|------|------|
| notes | 笔记列表数据 |
| currentNote | 当前选中的笔记 |
| currentPanel | 当前显示的面板 |
| searchQuery | 搜索关键词 |

---

## 六、边界情况

| 场景 | 处理 |
|------|------|
| 空输入 | 禁用发送按钮 |
| 网络错误 | 显示错误提示 |
| 搜索无结果 | 显示空状态 |
| 删除确认 | 二次确认对话框 |
